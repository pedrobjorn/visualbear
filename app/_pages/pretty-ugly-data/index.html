Pretty Ugly Data Displaying Data in a minimal way

This project uses real world data to create colourful, abstract forms.

Developed over the span of one week, the objective was to try to take important, dry data and give it new life.

This is the first of many projects that hope to take otherwise boring data and present it in a more interesting and engaging manner.

Hint: Don't move the mouse to hide information
